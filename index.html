About this layout

First, thank you for showing interest in this layout. This layout is called 'springtime'. I designed it specifically so others may use it. Just please make sure not to remove the credit in the footer.

In this layout dividers, icons and the header are included in the assets folder. All graphics were also designed by me, and as such you may use them. Or ofcourse replace them with your own! If you want to replace the header, I suggest a size of 900x400px.

The main content section is a grid of four columns divided into articles.

Depending on which size you want to use adjust the class of the article in the HTML. The same goes for selecting a particular colourscheme. You can make it as colourful or as simple as you want by changing the class.

For example, this article has as class "full matcha", so that it spans the full width and its background is set to the matcha colour.

At the top of the CSS file I have put the variables for all the colours used. So you can easily change them if you'd like that without having to spit through all the code.

Please note that, although I do enjoy making websites as a hobby, I am not a professional. So things may not have been coded optimally. But making these free layouts do help me practice more, so it's a win-win right?
